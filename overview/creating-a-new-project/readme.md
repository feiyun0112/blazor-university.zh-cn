# 创建项目
- 打开 Visual Studio。
- 单击**创建新项目**。
- 在搜索框中键入 Blazor。
- 选择 **Blazor WebAssembly 应用**。
- 点击**下一步**。
- 输入项目名称，例如 **MyFirstBlazorApp**。
- 点击**下一步**。
- 选中 **ASP.NET Core 托管**复选框。
- 单击**创建**。
